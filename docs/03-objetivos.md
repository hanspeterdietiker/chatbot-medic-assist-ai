# Objetivos

## Objetivo geral

Criar um chatbot capaz de apoiar a triagem hospitalar inicial por meio da coleta guiada de dados do paciente e da classificacao automatizada de area recomendada e nivel de urgencia.

## Objetivos especificos

- Criar um chatbot capaz de coletar dados iniciais do paciente de forma guiada.
- Reestruturar um dataset hospitalar para treinar um modelo de classificacao.
- Prever ou sugerir a area hospitalar recomendada para cada caso.
- Classificar o nivel de urgencia em categorias como baixa, prioritaria ou emergencia.
- Gerar um resumo simples para apoiar recepcao, triagem humana ou atendimento medico.
- Manter o limite etico de nao diagnosticar doencas e nao prescrever medicamentos.
