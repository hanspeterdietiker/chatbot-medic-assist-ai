# Contextualizacao do problema

Redes hospitalares lidam com grande volume de pacientes, diferentes condicoes clinicas, especialidades e niveis de urgencia. Muitas vezes, o paciente nao sabe se deve procurar pronto-socorro, clinica medica, cardiologia, ortopedia, pediatria ou outra area.

Essa incerteza pode gerar:

- espera desnecessaria;
- sobrecarga na recepcao;
- encaminhamento inadequado;
- demora na identificacao de casos prioritarios.

O projeto propoe um chatbot para apoiar a triagem inicial. A partir de informacoes como idade, genero, condicao principal e dados basicos do atendimento, o sistema utiliza um modelo de classificacao para recomendar uma area hospitalar e indicar um nivel de urgencia.

A decisao final continua sendo humana e medica.
