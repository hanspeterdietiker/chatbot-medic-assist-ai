# Limite etico e seguranca

O Medic Assist AI deve ser apresentado como um sistema de apoio a decisao. Ele nao substitui avaliacao medica, nao prescreve medicamentos e nao define diagnostico.

As respostas do chatbot devem ser formuladas de maneira segura, evitando afirmacoes clinicas conclusivas.

O sistema pode:

- coletar informacoes iniciais do paciente;
- sugerir uma area hospitalar para encaminhamento;
- indicar um nivel de urgencia;
- gerar um resumo para triagem humana.

O sistema nao pode:

- confirmar diagnosticos;
- prescrever medicamentos;
- substituir atendimento medico;
- tomar decisoes clinicas finais sem validacao humana.

A validacao final deve permanecer com profissionais de saude.
